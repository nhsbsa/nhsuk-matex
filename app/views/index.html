Maternity Exemption prototype

Goal

This prototype will contain multiple iterations of solutions to provide a simple to use digital maternity exemption application.

Since April 2022, the UCD team have revisted the service to improve the user journey so that it better aligns with GDS standards and to improve uptake of the digital service.
